# 중고서적 거래 웹앱

## #실행 환경
1. Node.js 설치 / https://nodejs.org/en/
2. MongoDB 설치 / https://www.mongodb.com/try/download/community

## #실행 방법
> npm install -g yarn

## 1. 서버 실행
```
1) cd server

2) yarn install

3) yarn start
```
## 2. 소켓 실행
```
1) cd socket

2) yarn install

3) yarn start
```
## 3. 클라이언트 실행
```
1) cd client

2) yarn install

3) yarn start
```
